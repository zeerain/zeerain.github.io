css的水平垂直居中到底有几种实现方式

发表于 2018-03-31   |   分类于 前端   |  

最近找工作，发现垂直水平居中的实现方式基本上百分百问的。网上资料参差不齐，有的说五种，有的说七八种。在此总结一下，到底有几种实现，顺便理解一下里面的实现原理及适用场景。

根据元素的宽高已知未知大致分为两类适用场景：

  1. 一类是已知宽高的元素如何垂直居中
  2. 一类是未知宽高元素的如何垂直居中

水平垂直居中主要的实现方式为以下6种：

  1. 利用绝对定位实现
  2. 利用flex布局实现
  3. 利用grid布局实现
  4. 利用line-height行高实现
  5. 利用table-cell实现
  6. 利用table布局实现

什么是水平垂直居中

首先我们先要明确一个概念，那就是我们一般所说的水平垂直居中，到底是什么意思？

  • 主体上至少有两个元素，父元素和子元素，子元素相对于父元素居中
  • 两个方向需要居中，水平居中+垂直居中

利用绝对定位的方式实现

利用绝对定位的方式，大约有以下四种实现方式。

绝对定位+margin

实现原理：父元素相对定位，子元素绝对定位。子元素top设置50%相当于往下移动父元素高度的50%,left设置为50%相当于往右移父元素宽度的50%，然后此时子元素的左上角就在父元素的正中心，利用margin-top和margin-left,把子盒子往上移动自身高度和宽度的一半，即可实现水平垂直居中。已知子元素的宽高

绝对定位+margi+calc

实现原理：和上面的那种实现方式原理都是利用绝对定位来实现的，但是这一次top和left的设置一次到位，利用了css的calc函数,父元素50%的宽度和高度再减去自身高度和宽度的一半，刚好位于正中心的位置。已知子元素的宽高

绝对定位+margin+auto

实现原理：子元素绝对定位，top、left、bottom、right分别设置为0，然后再利用margin: auto,此时的auto,其实是相当与上下左右都auto，所以也可以实现水平垂直居中。未知子元素宽高

绝对定位+tranforms

实现原理：也是和上面一样，利用绝对定位来垂直水平居中，但是这里利用css3的transform的translate属性来基于自身宽度和高度来进行位移。未知子元素宽高
